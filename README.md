# RITUAL

A 2-player strategic card game with occult theming. Players draft a deck from a shared pool, then duel to complete the ritual before their opponent.

---

## How the Game Works

RITUAL is played in two phases:

**Phase 1 — The Winston Draft**
Players build their decks by taking turns drafting from 3 shared piles. Pass a pile and it grows; take a pile and it resets. Draft ends when the draw stack is empty.

**Phase 2 — The Duel**
Players race to empty their deck. First player to attempt a draw with 0 cards remaining wins. If your Sanity reaches 0 at any point, you lose immediately.

**Resources**
- **Essence** — resets each turn, generated by Essence cards on your battlefield. Used to play Minions, Actions, and Devices.
- **Sanity** — starts at 10, never resets. Spent to play Forbidden Action cards. Hit 0 and you lose.

## Factions

| Faction | Color | Identity |
|---|---|---|
| **Gilded** | Gold | Sacrifice synergies, death triggers |
| **Oblique** | Gray-blue | Hand manipulation, information, bounce |
| **Infernal** | Dark red | Discard synergies, damage |
| **Wretched** | Green | Graveyard disruption, minion flooding |
| **Forbidden** | Purple | Sanity costs, banish effects |
| **Device** | Teal | Neutral artifacts, Essence generation |

56 unique cards, 100 total in the shared draft pool.

## Prototypes

Both prototypes are **single-file HTML** — open in any browser, no build tools or dependencies required. Commissioned card artwork is base64-embedded in the playable builds.

### Winston Draft (`ritual-winston-draft-v1.html`)
- Full 100-card pool, all 56 unique cards with artwork
- Complete Winston Draft mechanics (3 piles, pass/draft flow, forced draw)
- Mobile-first layout
- Draft Complete screen exports human-readable deck lists + machine-readable JSON for duel integration

### Duel (`ritual-duel-v1.html`)
- Hotseat 2-player on same device
- Full turn structure: Ready → Draw → Main → End
- All resource management: Essence pool, Sanity tracking
- 26 of 56 card effects implemented (v1)
- Picker modal for interactive choices (discard, scry, banish, target)
- Pass screen for hotseat handoff between players
